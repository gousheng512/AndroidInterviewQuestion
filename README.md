#### 1.如何将自己的md文件展示出来

答：将自己的md文件上传到trunk根目录，或者根目录的二级目录里即可。上传成功后，即可在页面上就能以wiki的形式展示出你的md内容。

运用MDWiki。MDWiki是 基于 Markdown 和 HTML 5 的纯客户端 Wiki/CMS
